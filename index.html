What is MVC?

MVC is short for Model, View, Controller, an architural paradigm used in many languages to organize files for a website's backend. Look below for some specifics!

So, what is the Model in MVC?

Model refers to any code pertaining to the database portion of a websites request. Any changes, updates, general requests, all that code is placed in the Model file. The model, or database, only speaks to the controller.

So, what is the View in MVC?

View refers to anything that the client will see, basic HTML, CSS, and Javascript, as well as images. This file will talk to the controller, or directly to the client, depending onthe language.

So, what is the Controller in MVC?

Controller refers to the middleman of the files, it connects the client/router to the database/model and to the view. Through the controller is the only way for the client and view to access the Model or database.
